# SOP: Updating Content and Resources

## What are content and resources?

Resources appear in the "Content" tab of the SDR Call Guide, organized by vertical. These are downloadable or linkable assets that SDRs can share with prospects or reference before a call.

**Supported content types:**
- Customer story PDFs
- One-pagers (vertical overviews, product summaries)
- Consensus or demo videos
- ROI calculators
- Product pages or external links
- Any other leave-behind or reference material

## When to use this SOP

- You have a new customer story PDF to add
- You have a new one-pager or leave-behind document
- You want to add a video link (Vidyard, Loom, YouTube, etc.)
- You want to add a link to an external page (product page, ROI calculator, etc.)
- You want to update the description or sub-vertical tag on an existing resource

## What you'll need to provide

Each resource needs 6 pieces of information:

### 1. Title

The customer name or asset name. Keep it short.

**Examples:**
- "Pacifica Hotels"
- "Ottimate for Hospitality - One-Pager"
- "AP Automation ROI Calculator"
- "Ottimate Platform Demo"

### 2. Type

What kind of asset this is. Current types:

| Type | When to use |
|---|---|
| case-study | Customer story PDFs |
| one-pager | Vertical overviews, product summaries, leave-behinds |
| video | Demo videos, consensus videos, testimonials |
| website | Product pages, calculators, external links |

### 3. Vertical

Which vertical tab this resource appears under in the app.

| Vertical | Use for |
|---|---|
| Hospitality | Hotels, restaurants, golf/country clubs, senior living |
| Grocery | Grocery chains, specialty food retailers |
| Healthcare | Dental, DSO, senior living, healthcare orgs |
| General | Cross-vertical resources, CAS/accounting firms, general Ottimate content |

A resource can appear under multiple verticals if relevant (e.g., a CAS firm that serves hospitality clients could appear under both Hospitality and General).

### 4. Sub-vertical

A more specific label that helps SDRs quickly identify relevance. This appears as a colored tag on the card.

**Sub-verticals currently in use:**
- Hotels
- Restaurants
- Golf/Country Clubs
- Grocery
- DSO
- Senior Living

You can use an existing sub-vertical or create a new one if needed.

### 5. Metric or description

One line that gives the SDR a reason to use this resource. For customer stories, pull a headline stat. For other content, write a short description.

**Examples:**
- "7,000+ invoices/mo, manual GL coding bottleneck" (customer story)
- "Overview of USALI compliance and multi-property GL mapping" (one-pager)
- "3-minute platform walkthrough for hospitality prospects" (video)
- "Interactive savings calculator by vertical" (website)

### 6. URL or file

- **For PDFs:** Attach the file. Your repo contact will add it to the app and set up the link.
- **For videos:** Paste the share link (Vidyard, Loom, YouTube, Vimeo, etc.)
- **For websites:** Paste the URL

## No AI tool needed

Resources are straightforward. Just fill out the template below and send it in.

---

**Template to copy and fill out:**

```
RESOURCE SUBMISSION

Title:
Type: [case-study / one-pager / video / website]
Vertical: [Hospitality / Grocery / Healthcare / General]
Sub-vertical: [e.g., Hotels, DSO, Restaurants, Golf/Country Clubs]
Metric or description:
URL or file: [Paste a link, or note that you're attaching a file]

Notes: [Any additional context, e.g., "This should also appear under General"]
```

---

**Submitting multiple resources at once?** Copy the template once per resource.
